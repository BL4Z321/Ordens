# 📌 Backlog organizado por módulos

## 1️⃣ Relatórios Impressos

Objetivo: controle e auditoria
Sugestões:

  - Ordem de Produção (PDF)
  - Relatório diário/mensal de produção
  - Consumo de materiais
  - Status das ordens
  - Tecnologias comuns: PDF (ex: dompdf, jsPDF)

Prioridade: 🟠 Média

## 2️⃣ Notificações Internas

Objetivo: agilidade e comunicação
Exemplos:

- Nova ordem criada
- Ordem atrasada
- Ordem finalizada
- Falta de material
- Canais:
  - Notificação no sistema
- (Futuro) E-mail / WhatsApp

Prioridade: 🟠 Média

## 3️⃣ Sistema de Compras

Objetivo: controle de materiais
Funcionalidades:

- Cadastro de fornecedores
- Solicitação de compra
- Aprovação
- Entrada de materiais no estoque
- Histórico de compras
- Integração: Estoque + Produção

Prioridade: 🔴 Alta (se produção depende de material)
